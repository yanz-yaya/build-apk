# Flutter APK Builder via GitHub Actions

Bot Telegram untuk build APK Flutter dari file ZIP menggunakan GitHub Actions.

## Cara penggunaan
1. Kirim file .zip project Flutter ke bot
2. Pilih mode Release/Debug
3. Tunggu 2-5 menit
4. APK siap download

## Setup
- Bot Telegram: @BotFather
- GitHub Token: Settings → Developer settings → Personal access tokens
